# Changelog — Ma Table (Admin)
All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- **Initial Scaffold**: Next.js 15 app with Tailwind CSS.
- **Admin Auth**: Pseudo:Password system via environment variables.
- **Restaurant Management**: List and edit (name, slug) existing restaurants.
- **Global Stats**: Overview of total restaurants, orders, users, and reservations.
- **Nova Connect Admin**: New dashboard section to manage AI matching parameters and Ollama clusters.
- **Schema Alignment**: Sync with MaTable-API schema for real-time data access.
